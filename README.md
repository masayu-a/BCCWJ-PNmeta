# BCCWJ-PNmeta

## Description
『現代日本語書き言葉均衡コーパス』の新聞記事データ (PN) の記事境界情報

## Features

1行目がヘッダです。

各列は以下の通りです。
- サンプルID
- 開始位置
- サンプル内記事番号
- 分類（大）
- 新聞名
- 掲載紙面
- 記事種別A
- 記事種別
- 内外
- 分類（大）
- 分類（小）
- 備考_1
- 備考_2
- SUW語数
